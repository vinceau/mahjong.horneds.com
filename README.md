Riichi Mahjong Scores Trainer
=============================

Based on the source files for https://mahjong.horneds.com

New changes reployed to: http://vince.id.au/mahjong.horneds.com/


## Changes (compared to the original)

* Improved support for mobile
* Improved random hand generation
  * All the yaku will now be generated so you can test your yaku knowledge as well as scoring


### Build Instructions

```
yarn install
yarn run build:coffee
yarn run production

# to deploy to github pages
yarn run deploy
```
